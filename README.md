# 📸 Image Watermarker Script

A simple and efficient Python script to automatically add a logo watermark to the bottom-center of multiple images. Perfect for website administrators and content managers.

## 🛠️ Features
- Processes images in bulk from an `input` folder.
- Automatically places the watermark at the bottom-center (with a 15px bottom margin).
- Resizes the logo to a standard 100x100px.
- Supports `.png`, `.jpg`, and `.jpeg` formats.
